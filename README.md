# Sistema-de-gestao-de-Funcionarios
Sistema de Gestão de Funcionários desenvolvido em Java Web como projeto acadêmico, feito por Esperança da Felicidade Roque Xavier, e Silva Atumane. A aplicação permite o gerenciamento de funcionários através de operações de cadastro, listagem, edição e exclusão de registros. Tecnologias utilizadas: Java, Servlet/JSP, HTML, CSS, Bootstrap e banco de dados MySQL.
